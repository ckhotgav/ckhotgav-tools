# 12-知識管理系統設定引導

> 💡 這個技能教你「怎麼用」Obsidian 整理知識。建議先裝好技能 11 再來。

## 三層結構：讓你的筆記不再亂七八糟

把你的 Obsidian 知識庫分成三層：

```
📁 你的知識庫/
├── 📥 收件匣/        # 第一層：臨時筆記
├── 📂 專案/          # 第二層：正在進行的事
└── 🏛️ 知識庫/        # 第三層：整理好的知識
```

### 第一層：收件匣 (Inbox)

放所有「臨時想到的東西」：
- 突然看到的重點
- 隨手記下的想法
- 還來不及分類的資料

### 第二層：專案 (Projects)

放你「正在進行的事情」，每個專案一個資料夾：
- 網站專案
- 學習筆記
- 工作任務

### 第三層：知識庫 (Knowledge)

放已經「整理好的知識」，可以永久參考：
- 技術筆記
- 觀念整理
- 教學文章

## 讓 AI 幫你整理

對你的 AI 編輯器說：
```
請在我的 Obsidian 知識庫中建立三層結構：
收件匣、專案、知識庫 三個資料夾。
並在專案中建立一個「我的學習筆記」資料夾。
```

## 建立筆記模板

讓 AI 幫你建立常用的筆記模板：

對 AI 說：
```
請在 Obsidian 中建立一個「日記模板」，內容包含：
- 日期
- 今天完成了什麼
- 遇到了什麼問題
- 明天的計劃
```

## 設定 AGENTS.md

在你的 Obsidian 知識庫根目錄建立一個 AGENTS.md 檔案，告訴 AI 你的知識庫規則：

對 AI 說：
```
請幫我在 Obsidian 知識庫中建立 AGENTS.md，
告訴 AI 我的知識庫有三層結構，
並且 AI 新增筆記時要先放在收件匣。
```

## 進階功能：每週知識重整

以後只要對 AI 說「**整理**」或「**週整理**」，AI 就會自動幫你執行以下流程：

### 步驟一：整理收件匣 + Clippings
- 讀取收件匣 AND Clippings 中所有筆記
- Clippings 是 Obsidian Web Clipper 存進來的網頁剪輯
- 問你哪些要保留、哪些可以刪
- 要保留的幫你搬到對應的專案或知識庫

### 步驟二：更新專案進度
- 檢查各專案資料夾中的筆記
- 問你最近的進度如何
- 更新專案狀態記錄

### 步驟三：清理無用筆記
- 找出超過 30 天沒動過的筆記
- 問你要封存還是刪除

### 步驟四：可選備份到 NotebookLM
- 如果你有裝技能 02（NotebookLM），可以把重要筆記送到 NotebookLM 做 AI 分析

### 設定方式

對 AI 說：
```
請幫我在 Obsidian 的 AGENTS.md 中加上「週整理」流程，
以後我只要說「整理」或「週整理」，
你就自動幫我執行收件匣整理、專案更新、清理、備份這四個步驟。
```

## 安裝 Chrome 擴充外掛（選用）

以下外掛可以幫你**從網路上快速擷取資料**，直接存進 Obsidian 或 NotebookLM。

### 外掛一：Kortex — NotebookLM 知識庫擷取

Kortex 是一個 Chrome 外掛，能用來收集網路文章，再送進 NotebookLM 分析。

**安裝方式：**
1. 打開 Chrome 線上應用程式商店，搜尋「Kortex」
2. 點「加到 Chrome」安裝
3. 安裝後右上角會出現 Kortex 圖示

**使用方式：**
- 看到有興趣的文章 → 點 Kortex 圖示 → 選擇儲存
- 內容會自動送到你的 NotebookLM 知識庫
- 適合用來收集研究資料、新聞、教學文章

### 外掛二：YouTube to NotebookLM

這個外掛可以把 YouTube 影片內容直接送到 NotebookLM，變成可分析的文字資料。

**安裝方式：**
1. 在 Chrome 線上應用程式商店搜尋「YouTube to NotebookLM」
2. 點「加到 Chrome」安裝
3. 安裝後在 YouTube 影片頁面會出現「Send to NotebookLM」按鈕

**使用方式：**
- 打開你想分析的 YouTube 影片
- 點下方的「Send to NotebookLM」按鈕
- 影片的逐字稿或摘要會自動匯入 NotebookLM

### 外掛三：Obsidian Web Clipper

這個是 Obsidian 官方出的 Chrome 外掛，可以**把網頁內容直接剪貼進 Obsidian 筆記庫**。

**安裝方式：**
1. 去 [Obsidian Web Clipper 官網](https://obsidian.md/clipper) 或 Chrome 商店搜尋
2. 點「加到 Chrome」安裝
3. 安裝後瀏覽器右上角會出現 Obsidian 圖示

**第一次使用設定：**
1. 點瀏覽器右上角的 Obsidian 圖示
2. 點「設定」→ 選擇你的 Obsidian Vault 路徑
3. 選擇筆記要存到哪個資料夾（建議存到「收件匣」）

**使用方式：**
- 看到想保存的網頁 → 點 Obsidian 圖示
- 可以選擇：擷取全文、摘要模式、或書籤模式
- 內容會自動以 Markdown 格式存入你的 Obsidian 收件匣
- 之後整理時對 AI 說「整理收件匣」即可分類
