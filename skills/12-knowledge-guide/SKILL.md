# 12-知識管理系統設定引導

> 💡 這個技能教你「怎麼用」Obsidian 整理知識。建議先裝好技能 11 再來。

## 三層結構：讓你的筆記不再亂七八糟

把你的 Obsidian 知識庫分成三層：

```
📁 你的知識庫/
├── 📥 收件匣/        # 第一層：臨時筆記
├── 📂 專案/          # 第二層：正在進行的事
└── 🏛️ 知識庫/        # 第三層：整理好的知識
```

### 第一層：收件匣 (Inbox)

放所有「臨時想到的東西」：
- 突然看到的重點
- 隨手記下的想法
- 還來不及分類的資料

### 第二層：專案 (Projects)

放你「正在進行的事情」，每個專案一個資料夾：
- 網站專案
- 學習筆記
- 工作任務

### 第三層：知識庫 (Knowledge)

放已經「整理好的知識」，可以永久參考：
- 技術筆記
- 觀念整理
- 教學文章

## 讓 AI 幫你整理

對你的 AI 編輯器說：
```
請在我的 Obsidian 知識庫中建立三層結構：
收件匣、專案、知識庫 三個資料夾。
並在專案中建立一個「我的學習筆記」資料夾。
```

## 建立筆記模板

讓 AI 幫你建立常用的筆記模板：

對 AI 說：
```
請在 Obsidian 中建立一個「日記模板」，內容包含：
- 日期
- 今天完成了什麼
- 遇到了什麼問題
- 明天的計劃
```

## 設定 AGENTS.md

在你的 Obsidian 知識庫根目錄建立一個 AGENTS.md 檔案，告訴 AI 你的知識庫規則：

對 AI 說：
```
請幫我在 Obsidian 知識庫中建立 AGENTS.md，
告訴 AI 我的知識庫有三層結構，
並且 AI 新增筆記時要先放在收件匣。
```

## ✅ 完成

你已經學會怎麼用 Obsidian 整理知識了！每天花 5 分鐘整理，知識會越累積越多。
