# 10-生成圖片 (Draw)

讓 AI 幫你畫圖！有兩種方式：新手用內建功能，進階用 API。

## 方式一：新手用內建生圖功能（最簡單）

如果你用的是 Antigravity 或 OpenCode Desktop，直接對 AI 說：
```
請幫我畫一張「一隻可愛的柴犬在沙灘上跑步」的圖片
```
AI 會自動用內建的生圖功能幫你生成圖片。

## 方式二：用 OpenAI API 生圖

如果你有 OpenAI API 額度，可以用更強大的生圖能力。

### 安裝工具
```powershell
npm install -g openai
```

### 讓 AI 幫你生圖
對你的 AI 編輯器說：
```
我有一個 OpenAI API 金鑰：[貼上你的金鑰]
請幫我寫一支程式，用 gpt-image-2 生成一張圖片，
主題是「夕陽下的城市天際線」，並存成 output.png。
```

## 小撇步

- 描述越詳細，AI 畫得越準確
- 可以指定風格：水彩、油畫、像素風、動漫風
- 可以指定比例：正方形、橫幅、直幅

## ✅ 完成

你現在可以讓 AI 幫你畫圖了！
