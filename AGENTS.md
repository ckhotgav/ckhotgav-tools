# ckhotgav-tools 全能工具箱

## 這是什麼

這是公開發布到 GitHub 的 AI 編輯器新手懶人包。
目標是幫完全不懂程式的新手，快速建立 AI 開發環境。

## 平台支援

以 Windows 為主撰寫。每個指令旁標註 macOS 對應寫法。
AI 應自動偵測使用者平台，給出對應指令。

## 重要原則

- 所有說明必須**新手友善**：用白話中文、避免程式術語、每一步解釋為什麼
- 每個技能都可以**獨立安裝**，使用者可以選裝部分技能
- 主文件 `點哥AI全能工具箱.md` 提供完整從頭到尾的流程
- skills 資料夾讓 AI 可以自動安裝
- Mac 使用者看到 `winget` 或 `PowerShell` 指令時，AI 要自動轉成 `brew` 或 `zsh`
